Выберите удобный для себя способ оплаты

Этот способ доступен для всех клиентов Aeroline, имеющих номер заказа. Для физических лиц оплата производится по согласованию нетто-стоимости. Вы можете воспользоваться СБП (системой быстрых платежей) и картами Visa, Mastercard, Maestro, МИР. Для юридических лиц этот вид оплаты доступен только с подготовленной карты.
